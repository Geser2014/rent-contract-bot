**7. Text-input states — handle_contract_date, handle_act_date, handle_monthly_amount, handle_deposit_amount, handle_phone, handle_email:**
All follow the same pattern (validator-call → isinstance check → reprompt or advance):

- **handle_contract_date**: calls `validate_date(update.message.text)`. On error: reply error + return `CONTRACT_DATE`. On success: store `context.user_data["contract_date"] = result`, reply `"Введите дату Акта приёма-передачи (ДД.ММ.ГГГГ):"`, return `ACT_DATE`.

- **handle_act_date**: calls `validate_date(update.message.text)`. On error: return `ACT_DATE`. On success: store `context.user_data["act_date"] = result`, reply `"Введите сумму ежемесячной аренды (например, 50000):"`, return `MONTHLY_AMOUNT`.

- **handle_monthly_amount**: calls `validate_amount(update.message.text)`. On error: return `MONTHLY_AMOUNT`. On success: store `context.user_data["monthly_amount"] = result`, reply `"Введите сумму депозита:"`, return `DEPOSIT_AMOUNT`.

- **handle_deposit_amount**: calls `validate_amount(update.message.text)`. On error: return `DEPOSIT_AMOUNT`. On success: store `context.user_data["deposit_amount"] = result`. Build deposit_method keyboard — read `deposit_amount` from user_data to compute half: `half = int(result / 2)`. Keyboard: two buttons `InlineKeyboardButton("Единовременно", callback_data="lump")` and `InlineKeyboardButton(f"50%+50% ({half}р + {half}р)", callback_data="split")`. Reply: `"Выберите способ внесения депозита:"`. Return `DEPOSIT_METHOD`.

- **handle_deposit_method**: `query = update.callback_query; await query.answer()`. Set `context.user_data["deposit_split"] = (query.data == "split")`. `await query.edit_message_text("Введите телефон арендатора (формат: +7 XXX XXX XX XX):")`. Return `PHONE`.

- **handle_phone**: calls `validate_phone(update.message.text)`. On error: return `PHONE`. On success: store `context.user_data["tenant_phone"] = result`, reply `"Введите email арендатора:"`, return `EMAIL`.

- **handle_email**: calls `validate_email(update.message.text)`. On error: return `EMAIL`. On success: store `context.user_data["tenant_email"] = result`, reply `"Отправьте *первую страницу паспорта* арендатора как файл (Прикрепить → Файл, не как фото):"` with `parse_mode="Markdown"`, return `PASSPORT_PAGE1`.

**8. Passport upload handlers:**

- **handle_passport_page1**: receives `filters.Document.ALL`. Download: `tg_file = await context.bot.get_file(update.message.document.file_id)` then `file_bytes = bytes(await tg_file.download_as_bytearray())`. Store `context.user_data["passport_page1"] = file_bytes`. Reply: `"Страница 1 принята.\nТеперь отправьте *страницу с пропиской* как файл (Прикрепить → Файл):"` with `parse_mode="Markdown"`. - **handle_passport_photo_warning_p1**: receives `filters.PHOTO`. Reply: `"Пожалуйста, отправьте фото паспорта как *файл* (Прикрепить → Файл), а не как фотографию — иначе качество будет недостаточным для распознавания."` with `parse_mode="Markdown"`. Return `PASSPORT_PAGE1`.

- **handle_passport_page2**: receives `filters.Document.ALL`. Download the same way as page1. Store `context.user_data["passport_page2"] = file_bytes`. Reply: `"Распознаю паспорт... ⏳"`. Pop large bytes from user_data after OCR: `context.user_data.pop("passport_page1", None); context.user_data.pop("passport_page2", None)`. Build summary: `unclear = ocr_service.get_unclear_fields(fields); summary = ocr_service.format_ocr_summary(fields)`. If unclear: append `f"\n\n⚠️ Нечитаемые поля: {', '.join(unclear)}"` to summary. Build confirm keyboard: `[[InlineKeyboardButton("Подтвердить ✅", callback_data="confirm"), InlineKeyboardButton("Переснять паспорт 🔄", callback_data="retry_passport")]]`. Reply: `summary` with `parse_mode="Markdown"` and keyboard. Return `CONFIRM`.

- **handle_passport_photo_warning_p2**: same warning text as p1. Return `PASSPORT_PAGE2`.

**9. Confirmation handler — handle_confirm:**
- `query = update.callback_query; await query.answer()`
- If `query.data == "retry_passport"`: pop `passport_page1`, `passport_page2`, `passport_fields` from user_data. `await query.edit_message_text("Отправьте *первую страницу паспорта* как файл:", parse_mode="Markdown")`. Return `PASSPORT_PAGE1`.

  - Read `ud = context.user_data; fields = ud["passport_fields"]`

  - Parse `passport_issued_date = validate_date(fields["passport_issued_date"])`. Same error handling — return `PASSPORT_PAGE1`.
  - Validate age: `age_ok = validate_age(tenant_dob, ud["contract_date"])`. If `isinstance(age_ok, str)`: `await query.edit_message_text(f"Ошибка: {age_ok}\nНачните заново: /start")`. Return `ConversationHandler.END`.
  - Generate contract number: `contract_number = generate_contract_number(ud["group"], ud["apartment"], ud["contract_date"])`
  - Construct ContractData with all fields from ud and fields dict. Every field must be provided — see ContractData interface above.

  - Log at INFO: `logger.info("ContractData assembled: contract_number=%s", contract_number)`

  - Return `ConversationHandler.END`

**10. /cancel handler — cmd_cancel:**
- `context.user_data.clear()`
- `await update.message.reply_text("Создание договора отменено. Для начала введите /start")`
- Return `ConversationHandler.END`

**11. Unexpected input fallback — handle_unexpected:**
- Reply: `"Не понимаю этого сообщения. Следуйте инструкциям бота.\nДля отмены введите /cancel"`
- Return `None` (stays in current state — do NOT return ConversationHandler.END)

**12. Factory function — build_conversation_handler:**
```python
def build_conversation_handler() -> ConversationHandler:
    return ConversationHandler(
        entry_points=[CommandHandler("start", cmd_start)],
        states={
            GROUP:          [CallbackQueryHandler(handle_group)],
            APARTMENT:      [CallbackQueryHandler(handle_apartment)],
            CONTRACT_DATE:  [MessageHandler(filters.TEXT & ~filters.COMMAND, handle_contract_date)],
            ACT_DATE:       [MessageHandler(filters.TEXT & ~filters.COMMAND, handle_act_date)],
            MONTHLY_AMOUNT: [MessageHandler(filters.TEXT & ~filters.COMMAND, handle_monthly_amount)],
            DEPOSIT_AMOUNT: [MessageHandler(filters.TEXT & ~filters.COMMAND, handle_deposit_amount)],
            DEPOSIT_METHOD: [CallbackQueryHandler(handle_deposit_method)],
            PHONE:          [MessageHandler(filters.TEXT & ~filters.COMMAND, handle_phone)],
            EMAIL:          [MessageHandler(filters.TEXT & ~filters.COMMAND, handle_email)],
            PASSPORT_PAGE1: [
                MessageHandler(filters.Document.ALL, handle_passport_page1),
                MessageHandler(filters.PHOTO, handle_passport_photo_warning_p1),
            ],
            PASSPORT_PAGE2: [
                MessageHandler(filters.Document.ALL, handle_passport_page2),
                MessageHandler(filters.PHOTO, handle_passport_photo_warning_p2),
            ],
            CONFIRM:        [CallbackQueryHandler(handle_confirm)],
        },
        fallbacks=[
            CommandHandler("cancel", cmd_cancel),
            MessageHandler(filters.ALL, handle_unexpected),
        ],
        name="contract_conversation",
        persistent=True,
        per_user=True,
        per_chat=True,
        allow_reentry=True,
    )
```
