Implement these functions:

**`_resize_image_bytes(raw_bytes: bytes, max_px: int = 1600) -> bytes`**
- Synchronous (wraps with to_thread at call site)
- `img = Image.open(io.BytesIO(raw_bytes))`
- `img.thumbnail((max_px, max_px), Image.LANCZOS)`
- `buf = io.BytesIO(); img.save(buf, format="JPEG", quality=92); return buf.getvalue()`

**`async def extract_passport_fields(page1_bytes: bytes, page2_bytes: bytes) -> dict[str, str]`**
- Resize both: `p1 = await asyncio.to_thread(_resize_image_bytes, page1_bytes)` and same for p2
- `page1_b64 = base64.standard_b64encode(p1).decode()`
- `page2_b64 = base64.standard_b64encode(p2).decode()`
- Call `_CLIENT.messages.create(model=_CLAUDE_MODEL, max_tokens=1024, system=_SYSTEM_PROMPT, tools=[_PASSPORT_TOOL], tool_choice={"type": "tool", "name": "extract_passport_fields"}, messages=[{"role": "user", "content": [{"type": "image", "source": {"type": "base64", "media_type": "image/jpeg", "data": page1_b64}}, {"type": "image", "source": {"type": "base64", "media_type": "image/jpeg", "data": page2_b64}}, {"type": "text", "text": "Первое изображение — главная страница паспорта (серия, номер, ФИО, дата рождения, место рождения, пол, кем выдан, дата выдачи, код подразделения). Второе изображение — страница с пропиской (адрес регистрации). Вызови инструмент extract_passport_fields со всеми 10 полями."}]}])`
- Iterate `response.content`; when `block.type == "tool_use" and block.name == "extract_passport_fields"`: log `input_tokens` and `output_tokens` at INFO, then `return block.input`
- If loop completes without returning: `raise ValueError(f"Claude did not return tool_use block. stop_reason={response.stop_reason!r} content={response.content!r}")`

**`def get_unclear_fields(fields: dict[str, str]) -> list[str]`**
- `return [k for k in PASSPORT_FIELDS if fields.get(k, "").strip().upper() == "UNCLEAR"]`

**`def format_ocr_summary(fields: dict[str, str]) -> str`**
- `lines = ["*Данные паспорта — проверьте внимательно:*\n"]`
- For each `(key, label)` in `_FIELD_LABELS.items()`: `value = fields.get(key, "UNCLEAR")`; if UNCLEAR append `" ⚠️ UNCLEAR"` to value; append `f"*{label}:* {value}"`
- Return `"\n".join(lines)`

**Anti-patterns to avoid (per research):**
- Do NOT use `anthropic.Anthropic()` (sync) — must be `AsyncAnthropic`
- Do NOT call `anthropic.AsyncAnthropic()` inside `extract_passport_fields` — use the module-level `_CLIENT` singleton
- Do NOT use prompt-only JSON — use `tool_use` with `tool_choice` forced
- Do NOT pass un-resized bytes — always resize first
