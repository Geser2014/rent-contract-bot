Critical details:
- `contract_data.pdf_path = pdf_path` must come BEFORE `save_contract` so the DB record

- `filename` uses `.replace('/', '_')` to make the contract number filesystem-safe for the
  Telegram document filename (Telegram rejects filenames with slashes).
- Do NOT re-edit the message after send_document — `query.edit_message_text` already showed
